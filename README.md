## Usage

1. Clone the repository:

```bash
> git clone https://github.com/Nicollas1705/mobile2you
```

2. Go to project root folder (mobile2you).

3. Run the app:

```bash
> flutter run
```

4. If necessary, run the following command before "flutter run":

```bash
> flutter pub get
```
